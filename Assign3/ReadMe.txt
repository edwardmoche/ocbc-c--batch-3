assignment 3 : Buat Rest API untuk Database Movie dimana API yang dibuat berfungsi menjadi 
penghubung ke database dengan method get| post| update| delete records dari dan ke - database MySQL

Cara penggunaan Aplikasi

- pertama run terlebih dahulu program dengan cara dotnet run
- jika sudah buka postman atau swagger dengan cara membuka localhost:[port yang ditampilkan pada terminal]/swagger/index.html
- selanjutnya jika ingin menambahkan data gunakan Post lalu masukan data sesuai dengan parameter yang dibuat
- kemudian untuk menampikan data yang telah diinput gunakan get

*note : untuk postman terlebih dahulu masukkan link localhostnya
